\doxysubsection{Detailed Description}
Static utility class for interference graph construction and register allocation. 

All allocation algorithms follow the same greedy graph-\/colouring approach:\+ a simplification phase removes nodes of degree \texorpdfstring{$<$}{<} K onto a stack, then a colouring phase pops them back and assigns the lowest available colour.
