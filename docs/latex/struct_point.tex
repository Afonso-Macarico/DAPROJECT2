\doxysubsection{Detailed Description}
Represents a single program point (line number) within a live range. 

A point may be a definition (is\+Def), a last-\/use/\+kill (is\+Kill), or a plain intermediate live point. A point can simultaneously be a def and a kill (e.\+g. i = i + 1).
